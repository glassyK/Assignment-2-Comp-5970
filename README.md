To accomplish the assignment:

Create a Repository:

Make sure it is a public repository.
then go to "code" at the top menu, then click on "create a new file" and add a file name " description" (you can add your description for this assignment here" , then click on "Commit Changes".
Go to Repository Settings:

In the left menu, choose Codespaces.
Select Setup Prebuild.
Choose “main” as the branch.
Start a Codespace:

Go back to your repository.
Click the Code button and select the Codespaces tab.
Follow the instructions in the assignment step by step.
Add an HTML File:

Create an HTML file and name it index.html.
Edit the file or add additional files based on the assignment instructions.
Set Up Live Server:

From the left menu, choose Extensions.
Search for the Live Server extension and install it.
Go back to the file directory, right-click on index.html, and select Open with Live Server to view the output.
Commit Your Changes:

In the left menu, select Source Control.
Review the changes you made and click the Commit button.
Confirm by selecting the OK button in the top-right corner of the editor.
Your changes will now be reflected in the main branch.
Handle Commit Errors (if any):

If you encounter errors during the commit process, an error message will appear.
Click the second button provided to view guidance.
If prompted to run a command, copy it and execute it in the Terminal tab.
Set Up GitHub Pages:

Go to Repository > Settings.
In the left menu, select Pages.
Adjust the settings by choosing the main branch.
You can also input a custom domain name if needed.
Activate Free Trial and Check Your Website:

Complete the steps for activating the free trial.
Return to Settings > Pages, where you’ll find the assigned link to your website.
Add /MyWebsite/index.html to the end of the URL to access your website.
Share the Link:

Follow the instruction of submitting the assignment on the Canvas and Share the website link  and developed code.
